# 00 · Start Here — interactive guide

An interactive, visual walkthrough of the 13 experiments in this repo — built for
learning by *poking things*, not reading walls of text.

**To open:** clone or download the repo, then double-click `index.html` (or drag it into
any browser). No server, no build, no internet needed — it's one self-contained file.

- Click the story map, or use the left sidebar, to jump to any experiment in any order.
- Every concept is an interactive toy first; the short text comes second.
- "Go deeper" panels stay collapsed until you want them.
- The emergence, dose-response, and unlearning charts are drawn from this repo's own
  result data (the `results/` CSVs), so you're exploring the actual experiments.

The "what's next" section links out to the `../11_induction_3d/` 3D viewers, so open this
file from inside the repo for those links to resolve.
